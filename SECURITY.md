# Security Policy for Kildear_Soc

## Reporting a Vulnerability

**Please DO NOT use public issues for security reports.**

Use GitHub's private vulnerability reporting:  
https://github.com/dwd-alt/Kildear_Soc/security/advisories

Expected response time: **48 hours**

## Security Measures Implemented

- ✅ All dependencies checked via Dependabot
- ✅ Static code analysis via CodeQL (enabled)
- ❌ No plaintext passwords stored
- ✅ SQLite database not exposed publicly
- ⚠️ Security audit planned (Open Source funding pending)

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Yes    |
| < 1.0   | ❌ No     |

## Responsible Disclosure

We follow standard responsible disclosure practices. Valid reports will be publicly acknowledged.
