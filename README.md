## 🚀 Быстрый старт

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Установка и запуск

```bash
# Клонируем репозиторий
git clone https://github.com/dwd-alt/Kildear_Soc.git
cd Kildear_Soc

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем приложение
python app.py
